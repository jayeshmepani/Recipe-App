# Recipe App

## Overview
The Recipe App is a Flutter application that allows users to view, search, filter, and sort a list of recipes. Each recipe includes a title, ingredients, instructions, and indicators for whether it is vegetarian or a drink.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites
Ensure you have Flutter installed on your machine. For more details, visit the [Flutter installation guide](https://flutter.dev/docs/get-started/install).

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/jayeshmepani/Recipe-App.git
   cd Recipe-App
   flutter pub get
   ```

2. **Connect a device or start an emulator:**
   ```sh
   flutter devices
   ```

   Example output:
   ```
   Found 4 connected devices:
     SM GXYZ1 (mobile) • EXAMPLE • android-arm64  • Android 13 (API 33)
     Windows (desktop) • windows     • windows-x64    • Microsoft Windows [Version 10.0.22635.3350]
     Chrome (web)      • chrome      • web-javascript • Google Chrome 126.0.6478.62
     Edge (web)        • edge        • web-javascript • Microsoft Edge 127.0.2651.8
   ```

3. **Run the app on a selected device/emulator:**

   For Android device:
   ```sh
   flutter run -d EXAMPLE
   ```
REPLACE "EXAMPLE" WITH ACTUAL ID

   For Windows desktop:
   ```sh
   flutter run -d windows
   ```

   For Chrome browser:
   ```sh
   flutter run -d chrome
   ```

   For Edge browser:
   ```sh
   flutter run -d edge
   ```

4. **Build the app:**

   For Windows:
   ```sh
   flutter build windows
   ```

   For Web:
   ```sh
   flutter build web
   ```
